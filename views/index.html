About AtishiBoT

AtishiBoT helps you build a better discord server, be it for you and your friends, or for an organisation or event. with a plathora of features, randing from moderation, to music, to your very own google engine, AtishiBoT is built to compliment the needs of every discord server out there.

With AtishiBoT, getting lyrics was never easier! You can search lyrics for any and every song out there! and here's the catch, you don't have to be listening to it ;)

Along with it's memes, roasts and inspiration features, AtishiBoT has got something very special for all the marvel fans out there! Use the earth616 command to find out which marvel character you are!

Commands

Prefix:   atishi

1  
atishi -h

Provides help commands to use AtishiBoT

Eg- atishi -h : Shows all help commands

2  
atishi hi

Say hi to atishi! You can make conversation with her after this ;)

Eg- atishi hi : Starts the chatbot

3  
atishi bye

Say bye to atishi! she doesn't like hearing it tho

Eg- atishi bye : Stops the chatbot

4  
atishi date

Get today's date

Eg- atishi date : Gives you the date

5  
atishi time

Get the current time

Eg- atishi time : Gives you the time

6  
atishi google *query*

Google anything from within discord

Eg- atishi google *query*: Returns the top most search result

7  
atishi inspire

Feeling down? Let atishi tell you something inspiring

Eg- atishi inspire: Tells you something inspiring

8  
atishi lyrics *song name*

Get the lyrics of a song! anytime you want

Eg- atishi lyrics Zaalima : Shows Lyrics of Zaalima

9  
atishi play *song name or link*

Play a song by entering a link or simply typing it's name

Eg- atishi play Zaalima : Plays Zaalima

10  
atishi stop

Run this command to stop playing a song

Eg- atishi stop: Stops playing the song

11  
atishi meme

Gets a meme from reddit

Eg- atishi lyrics Zaalima : Shows Lyrics of Zaalima

12  
atishi eart616

Find out which marvel character you are

Eg- atishi earth616 : Assigns a marvel card to you

13  
atishi roast *@user*

Roast someone with this command

Eg- atishi roast @user : atishi sends a roast to the user

24  
atishi simpmeter *@user*

Let atishi predict how big a simp someone is

Eg- atishi simpmeter @user : @user is 97% simp

14  
atishi ban *@user*

Moderator command to ban user from server

Eg- atishi ban @user : User gets banned from the server

15  
atishi kick *@user*

Moderator command to kick user from server

Eg- atishi kick @user : User gets kicked from the server

16  
atishi createRole *role name*

Moderator command to create a role

Eg- atishi createRole DJ : A new role `DJ` is created

17  
atishi deleteRole *role name*

Moderator command to delete a role

Eg- atishi deleteRole DJ : A new role `DJ` is deleted

18  
atishi assignRole *role name* *@user*

Moderator command to assign a role

Eg- atishi assignRole DJ @user : Role is assigned to all the users mentioned

19  
atishi removeRole *role name* *@user*

Moderator command to remove a role for a user

Eg- atishi removeRole DJ @user : Role is removed for all the users mentioned

20  
atishi createTC *channel name*

Moderator command to create a text channel

Eg- atishi createTC general : A text channel named general is created

21  
atishi deleteTC *channel name*

Moderator command to delete a text channel

Eg- atishi deleteTC general : A text channel named general is deleted

22  
atishi createVC *channel name*

Moderator command to create a voice channel

Eg- atishi createVC general : A voice channel named general is created

23  
atishi deleteVC *channel name*

Moderator command to delete a voice channel

Eg- atishi deleteVC general : A voice channel named general is deleted
